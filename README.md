## Highlighted Projects

A few repositories that reflect how I think and learn:

- [SoloStyle](https://github.com/owuorviny109/SoloStyle) – E-commerce experiment for premium footwear with React & TypeScript on the frontend and an Express backend, exploring payment flows and integration (M-Pesa STK push).

- [cloud-portfolio](https://github.com/owuorviny109/cloud-portfolio) – A cloud-native portfolio built with React, TypeScript, and Vercel, experimenting with performance, accessibility, and conversational UI.

- [ai-photo-gallery](https://github.com/owuorviny109/ai-photo-gallery) – Serverless photo gallery using AWS services and AI image analysis to understand event-driven flows and async processing.

- [terraform-aws-web-tier-infra](https://github.com/owuorviny109/terraform-aws-web-tier-infra) – Terraform configs for a basic web-tier architecture on AWS, used as a learning ground for IaC and AWS networking.

(These are learning projects, not production systems, but they’re where I test ideas about architecture, reliability, and operations.)
